# Update
**NASCAR Panel — v1.3**

https://high-voltage-studios.github.io/nascar2026/

**New Features**

**RC Sub-Roles** — Race Control selects a sub-role after login, each with its own password and scoped interface: Race Director (full override authority), Technical Inspector (inspection queue), Timing & Scoring (points only, no flags), Communications Director (radio and announcements only).

**Scorer / Official** — New role for race officials. Verify stage results, add official notes, lock results to prevent further edits, and export a printable results report.

**Driver Polls** — RC posts live polls visible to all roles. Racers vote from the Hub with real-time vote bars. RC controls who can vote and can close the poll at any time.

**Steward Decisions** — RC posts formal rulings that appear as a highlighted card in every role’s Hub, the incident feed, and the public page.

**Public Race Page** — No-login summary page showing current flag, standings, recent incidents, and steward decisions. Auto-refreshes every 30 seconds.

**Hub — All Roles** — Every role now has a Hub button. Opens as a full overlay with tabs for Schedule, Rulebook, Commentary, Team Radio, Polls, and Rulings.

**Changes**

**Network Administration** — All system configuration moved here: passwords, role access, lockout, TV code, and sub-role passwords. Also added a broadcast tool to push a full-screen alert to any role or all users.

**RC Admin** — Now contains only quick operational settings. All system config is in Net Admin.

**Login Screen** — Marshal, Racer, and Spectator are the primary cards. RC, Scorer, and Net Admin sit in a compact row below a divider.

**Bug Fixes**

- Hub overlay was hidden behind the Spectator screen
- Hub Polls and Rulings tabs were not responding to clicks
- RC announcements were only reaching Racers, now reach all roles
- Net Admin broadcast showed the wrong header on the overlay
