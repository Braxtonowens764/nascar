🚀 Setup Guide — Live Sync + Netlify
Follow these steps to go live. Takes about 5 minutes total.
Step 1 — Firebase (live sync)
1
Create a Firebase project
Go to console.firebase.google.com → Add project → name it (e.g. nascar-amr) → Continue → Disable Google Analytics → Create.
2
Create a Realtime Database
In your project: Build → Realtime Database → Create database → Choose a region → Start in test mode → Enable.
3
Get your config
Project Settings (gear icon) → Your apps → Web icon (</>) → Register app (any name) → Copy the firebaseConfig object.
4
Paste config into the file
Open index.html in any text editor. Find the line that says // For Firebase JS SDK v7.20.0 and later, measurementId is optional const firebaseConfig = { apiKey: "AIzaSyAa3C4jRMN3mtFKLVoeBWhoMTXR97RPWec", authDomain: "nascar-f03ec.firebaseapp.com", databaseURL: "https://nascar-f03ec-default-rtdb.firebaseio.com", projectId: "nascar-f03ec", storageBucket: "nascar-f03ec.firebasestorage.app", messagingSenderId: "1058635274597", appId: "1:1058635274597:web:9ecec006238fb6c165a98c", measurementId: "G-MYS4CCMJ1P" };
Step 2 — Netlify (free hosting)
5
Go to Netlify
Go to netlify.com → Sign up or log in with email or GitHub (free). No credit card needed.
6
Deploy your site
From the Netlify dashboard: click Add new site → Deploy manually → drag and drop your index.html file into the upload box → Netlify will deploy it instantly.
7
Your site is live
Netlify gives you a free URL like https://your-site-name.netlify.app. To use a custom name: Site settings → Domain management → Options → Edit site name.
8
Share with your team
Everyone opens that Netlify URL on their own device. Race Control on a laptop, Marshals on phones or tablets, Racers on theirs. All devices sync live through Firebase automatically.
Note on pastSessions: Completed race sessions are saved locally on each device only (not synced) to keep the database small. Live incident data, flags, drivers, and protocols all sync in real time.
